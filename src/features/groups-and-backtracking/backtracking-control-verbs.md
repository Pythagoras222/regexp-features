# description
A <dfn>Backtracking Control Verb</dfn> is a special pattern usually in the form of <code>(\*<em>VERB</em>)</code> or <code>(\*<em>VERB</em>:<em>arg</em>)</code> that performs some special behavior with respect to backtracking.
